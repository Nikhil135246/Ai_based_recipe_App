# 🍳 AI Recipe App

An Android recipe app with AI-powered cooking assistance. Search recipes, browse categories, and get cooking tips using Gemini AI.

## ✨ Features

- 🤖 AI recipe suggestions & cooking tips
- 🔍 Smart recipe search
- 📂 Browse recipes by categories
- 🍽️ Detailed ingredients & instructions
- ⭐ Popular recipes section
- ⚡ Fast and clean UI

# 🛠️ Tech Stack

- **Language:** Kotlin
- **Database:** Room DB
- **Networking:** Retrofit
- **AI:** Gemini API
- **Architecture:** Activity + Adapter pattern
- **UI:** XML Layouts

---

# 🚀 Installation

```bash
git clone https://github.com/Nikhil135246/Ai_based_recipe_App.git
cd Ai_based_recipe_App
```

Open in Android Studio and add your Gemini API key.

Run:

```bash
./gradlew assembleDebug
./gradlew installDebug
```

# 📱 Supported

- Android 7+ (API 24)
